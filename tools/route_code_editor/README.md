# Route Code Editor

### REQUIREMENTS
- Python 3.6.x
- Node 10.x


### SETUP

```
# Install Package
$ apt-get install vim git python-pip

# Install requiremtns
$ pip install django

# Install npm package
$ npm install

# build(development)
$ npm run build
# build(production)
$ npm run build:prod
```

### TEST
```
$ npm run test
```
