## Running

* Please pre-execution mosquitto

launch flask server
```
src$ python router.py
```

launch vehicle
```
src$ python vehicle.py vehicle1
```

launch fleetManager
```
src$ python fleetManager.py
```

launch trafficSignal
```
src$ python trafficSignal.py 599
```

launch user
```
src$ python user.py user1
```
